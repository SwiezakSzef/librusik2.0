Librusik
A cool Librus Synergia web client.
Website: dani3l0.github.io/librusik


The idea

The main idea was to check school grades without logging in Synergia each time you open the browser. After few hours of Librus API research it turned out that an almost complete replacement can be created... So, here it is :)))


Features

Librusik not only supports basic features like grades, timetable, free days, lucky numbers or exams, but also includes extra features including Librus' paid ones. Those are viewing messages with attachments, enforced average calculation (works even if school disables it), cool year countdown gauges and many more.


Personal data

You don't have to worry about your personal data. Synergia passwords are stored securely. Admin can only see your full name to identify who's using Librusik.
